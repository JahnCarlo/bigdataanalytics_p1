Big Data Analytics Project 1

Made by Jahn Marrero

This Project covers the basics of using mapreduce jobs through Hadoop DFS and implementation of such jobs to analyze data. The data analyze was a 100,000 tweets file. In these tweets, various keywords and various things were analyzed. See below visualizations to understand the analysis that was done. Visualizations were made using the D3.js library, using the Vida.io and Plotly frameworks.

Visualizations

  • Words To Find
  • Top Frequency of Words
  • Top Unique Names that Tweeted
  • Replies Network
  • Top Users that Tweet with TweetCount
